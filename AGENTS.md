# AGENTS.md

> for any agent (human or otherwise) committing to this repo.

## commit format

```
[XX] X#####
```

- `XX` = two random uppercase letters
- `X#####` = literal `X` followed by 5 random digits
- no semantic meaning. nothing decodes. don't try.
- one line only. no body. no subject. no scope.
- the diff is the description.

examples:

```
[AX] X89213
[QM] X40291
[BR] X73091
[ZK] X10487
```

## that's it

don't add conventional commits.
don't add prefixes.
don't add changelogs.
don't try to be clever.

if a commit needs explaining, edit the file directly. files explain
themselves. commits don't.
